IV. Others

border

Several properties that control the border around an element, including:

  • border-color : color of the border
  • border-style : style of the border
  • border-width : thickness of the border
HTML

      <p class="border_demo_detailed">This paragraph has a red border.</p>

      <p class="border_demo_shorthand">This paragraph looks the same but use a shorter css syntax.</p>
CSS

      .border_demo_detailed{
          border-color: red;
          border-style: solid;
          border-width: 2px;
      }

      /* all 3 properties can be merged into the shorthand property 'border': */
      .border_demo_shorthand{
        border: 2px solid red;
      }
Result

This paragraph has a red border.

This paragraph looks the same but use a shorter css syntax.

display

Set the display type of an element. Common values are:

  • display: inline: the element is added to the current line of the document as if it's a text. <a> and <span> are common html tags that use this display type as default.
  • display: block: the element take an entire line for itself. All content after it will be pushed to the next line. You can set the width and height of the element with CSS' width and height properties. <p>, <div> and <h1> are common tags that use this display type as default.
  • display: inline-block: an inline element whose width and height can be set with CSS properties. <img> comes with this display type as default.
  • display: none: and invisible element. It is not rendered by the browser and the user can't see it.
